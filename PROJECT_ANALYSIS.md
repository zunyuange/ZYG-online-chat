# 在线客服聊天系统 - 项目分析报告

## 一、项目概述

本项目是一个功能完整的**实时在线客服聊天系统**，基于 React + Hono 构建，支持本地开发和 Cloudflare Workers 边缘部署。系统采用多租户架构，支持商家隔离，每个商家可以拥有独立的客服团队和聊天数据。

### 项目定位
- **目标用户**: 企业客服团队、在线客服外包服务提供商
- **核心价值**: 提供高效的实时客户沟通能力，支持多商家隔离和客服团队管理

### 3.4 多租户隔离机制

```
staff_users 表设计:
┌─────────┬──────────────┬───────────────┬───────┐
│   id    │ business_id  │ business_slug │ role  │
├─────────┼──────────────┼───────────────┼───────┤
│    1    │      0       │   "default"   │ admin │  ← 商家主账号 (business_id=0)
│    2    │      0       │  "zunyuange"  │ admin │  ← 商家主账号
│    7    │      2       │               │ staff │  ← 归属商家2的客服
└─────────┴──────────────┴───────────────┴───────┘
```

- **商家主账号**: `business_id = 0`，拥有完整管理权限
- **客服账号**: `business_id > 0`，归属到特定商家，权限受限
